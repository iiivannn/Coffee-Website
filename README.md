# Cozy Grounds

A responsive single-page website for a coffee shop, built as a frontend project. It features the shop's menu, specialties, blog, and overall brand identity.

Warm aesthetic, smooth interactions, and reliable responsive behavior across all screen sizes.

Built using HTML, CSS, and JavaScript.

**[Live Website Demo](https://coffee-website-seven-indol.vercel.app/)**

<br>

## Tech Stack

| Layer     | Technology          |
| --------- | ------------------- |
| Markup    | HTML5               |
| Styling   | CSS3                |
| Scripting | JavaScript ES6+     |
| Filtering | MixItUp 3           |
| Icons     | Boxicons            |
| Fonts     | Google Fonts - Lato |

<br>

## Features

- Fixed navbar with scroll-aware background and active section highlighting
- Interactive product grid with animated category filtering
- Custom page loader with a coffee animation via gif
- Scroll-to-top button for added user experience
- Smooth scrolling across the website
- Mobile-first layout, fully responsive for all mobile devices up to large desktops

Built with a user-centric approach. Every layout and interaction is designed so users can navigate the site naturally and effortlessly.
